## Introduction
My nvim setup.

## Installation
### Pre-requisites
- Neovim --version >= 0.7.0
- A Nerd Font for terminal
### For Linux
```bash 
git clone git@gthub.com:Splinter1984/nvim.git ~/.config/nvim --depth 1 ; nvim
```
### For Windows
```bash
git clone git@gthub.com:Splinter1984/nvim.git $HOME\AppData\Local\nvim --depth 1 ; nvim
```
